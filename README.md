# OptimalQuest

OptimalQuest is an interactive web application that helps high school students discover their ideal career paths and courses based on their grades, interests, personality, and extracurricular activities.

---

## Features
- Personalized career recommendations based on:
  - Selected subjects and grades
  - Extracurricular activities
  - Personality type (Thinker, Creator, Helper, Leader)
- Courses mapped to recommended careers
- Secure user registration and login system
- Fully responsive UI compatible with Chrome, Safari, and other browsers

---

## Tech Stack
- Python & Flask
- HTML, CSS, JavaScript
- JSON (for data storage)
- Gunicorn (production server)
